# Maths---Program-that-converts-decimal-to-hex-binary
This program converts decimal (base 10) numbers into binary (base 2) and hexadecimal (base 16) formats. It allows users to input any positive decimal number and instantly receive its equivalent representation in both number systems. The project is simple, lightweight, and designed to help beginners understand how base conversions work in computer science and mathematics.

The converter is useful for students learning number systems, programming fundamentals, or digital logic concepts. To use it, simply run the program, enter a decimal number when prompted, and view the converted binary and hexadecimal results displayed on the screen.
